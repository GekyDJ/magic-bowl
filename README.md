# Colorful Glowing Liquid Bowl

An interactive and responsive simulation of a glowing liquid, built with pure CSS and Vanilla JavaScript. It features smooth gravity animations, realistic inertia (sloshing) effects, and a dynamic particle generator adapted for mobile devices.

**Live Demo:** [watch the live animation here](https://gekydj.github.io)

## Features

* **Realistic Fluid Dynamics:** Optimized inertia (sloshing) effect using synchronized CSS keyframes to simulate liquid weight and gravity.
* **Dynamic Bubble System:** A Vanilla JavaScript generator that creates random rising bubbles with varying sizes and speeds.
* **Mobile-First Optimization:** Responsive script that detects screen width and automatically scales bubble density and size for better visibility on mobile devices.
* **Clean & Synchronized Code:** Precise 10-second animation cycles with a "zero-leak" design to prevent visual artifacts (black gaps) during rotation.
* **Glow & Glassmorphism:** High-fidelity visual style using CSS filters, `drop-shadow`, and transparency for a modern neon look.
